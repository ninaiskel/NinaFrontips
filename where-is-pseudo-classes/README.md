# Pseudo classes :Where() & :is()

`:Where()` & `:is()` son unas pseudo classes de CSS que nos permiten escribir selectores compuestos de manera mas concisa, para evitar la repetici√≥n.

Ambos son funciones que toman una serie de argumentos de selectores como(ids, classes, tags etc) para luego pasarles el estilo que se desee.

## Syntax

```
:where(.class, p, ...){}
:is(a, .class, h4 ...){}
```
